this is the repo to my home assistant calendar one pane of glass dashboard for me and my family.
The aim:
a sidebar menu with different actions down it. first important ones are:
Main View
todo list
shopping list
home controls

A top bar showing amount of events and todo items left for the day per person

a floating circle plus button at the bottom right corner this will give you the option to add
a calendar event from the calendar
a todo item from the todo page
a shopping item from the shopping page
and invisible on the home controls page

the menu items above will go to each of their own screens
Main View:
    This should have a set of views toogleable:
    Day View
        this will show people the day and colour coded events.
    Week View
        this will show people the day and colour coded events.
    Month View
        on each day it will show a pip with the amount of events per person just the persons colour.
        if a person has zero it will not show their colour.

Todo List:
    This will show all people and then they should be toggleable to see the persons todo items.

Shopping List:
    This would be a todo style list but only for shopping so no due dates, sub tasks etc

home controls will just be a set of home controls for the house and can be done later but should be visible on the sidemenu to remind me it needs to be done later.
