# ha-family-board

Home Assistant configuration and custom dashboard for a family household.

Goals:
- One shared family calendar
- Simple chores and shopping lists
- Clear home controls
- Accessible on wall display, TV, tablet, and phone
- Minimal cognitive load (ADHD/autism friendly)

This repo tracks:
- Home Assistant configuration authored by us
- Custom Lovelace card for the family board

This repo does NOT track:
- Home Assistant runtime/state
- Secrets
- HACS or downloaded frontend cards
